# 🎧 SUR – Feel the Music Together

**SUR** is a next-generation music streaming web application that combines music playback with social interaction and immersive UI design.

🚀 Built with a modern tech stack, SUR delivers a smooth, stylish, and engaging music experience similar to Spotify — but with a unique futuristic UI.

---

## 🌟 Features

### 🎵 Music Player

* Play, pause, next, previous controls
* Interactive progress bar
* Volume control
* Smooth playback experience

### 🔍 Search System

* Search songs dynamically
* Fast and responsive UI
* YouTube-based music integration

### 📂 Library

* View saved or recent songs
* Organized music browsing

### 👥 Social Features (Planned / In Progress)

* Rooms for shared listening
* Real-time chat system
* Sync music with friends

### 📊 Insights (Upcoming)

* Listening stats
* User behavior tracking

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/sunnyx-dev/Music-Player.git
cd Music-Player
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the development server

```bash
npm run dev
```

App will run on:

```
http://localhost:5173
```

---

## 🔑 Environment Variables

Create a `.env` file and add:

```
VITE_YOUTUBE_API_KEY=your_api_key_here
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
```
